mnoGoSearch

mnoGoSearch - это пара программ для организации поиска в небольших сетях, а именно краулер, обходящий документы сети и сохраняющий информацию о найденных в них словах в базе данных SQL, а также приложение CGI, выполняющее поиск документа по записям в базе данных, сделанным индексатором.

Поисквые лунчеры

1.Kupfer является свободным, легким, простым в использовании инструментом, который позволяет быстро и легко получить доступ к приложениям и их документам. Цель разработки данного проекта довольно простая и понятная; найти приложение и запустить его. Он написан на языке Python и быстро ищет все ваши приложения и документы на вашем компьютере, и запускает их. Это приложение довольно много экономит времени у пользователей Linux для настольных компьютеров, чтобы повысить их производительность и ускорить свою ежедневную работу.


2.Установить Synapse в Ubuntu/Linux mint

Synapse представляет собой мощное приложение для Ubuntu Linux, которое позволяет запускать приложения, а также поиск и доступ к соответствующим документам и файлам путем использования Zeitgeist двигателя.

3. Установить Albert в Ubuntu/Linux mint

Как вы знаете, Albert как и все приложения для Linux, поставляется с открытым исходным кодом для запуска Linux-приложений, он написан на C++ и Qt5, с функциями, похожими на Synapse. Пользователь способен настроить горячие клавиши для открытия и запуска программ, а также может легко найти программное обеспечение, с помощью Searchbar. Он может также выполнять поиск в Google и среди Chrome-закладок.

проUlauncher – очень быстрый поисковый лаунчер для Ubuntu/Linux mint

4. Особенности Ulauncher

  • Поиск и запуск установленных приложений
  • Просмотр директорий и открытых файлов
  • Запуск Google и Wikipedia поиска
  • Клавиша быстрого доступа по умолчанию, для открытия приложения является Ctrl+Space.

Ulauncher очень быстрый и супер отзывчивый, отображает результат поиска мгновенно. Это действительно так, даже тогда, когда вы носитесь по папкам для поиска файлов. Благодаря нечеткому поиску вам даже не нужно беспокоиться о написании имени приложения или папки точь-в-точь, он и так все вам подскажет из его результатов.

Ulauncher поставляется с двумя встроенными плагинами: результатом поиска Google и поиск Wikipedia. Они просты в использовании: для использования поиска в Google, вам нужно просто добавить букву “g” в начале поиска, и соответственно букву “w”, для поиска в Wikipedia.
